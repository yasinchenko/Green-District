# MVP Scope

## Phase 1: Core Simulation (Current)

### Must Have
- ✅ SimulationClock (time management)
- ✅ WorldState (central data repository)
- ✅ UpdateManager (predictable update order)
- ✅ Citizen (basic agent)
- ✅ Basic needs system
- ⏳ Districts and territory
- ⏳ Economy system (jobs, income, businesses)

### Nice to Have
- UI visualization
- Event system
- Save/Load mechanics

## Phase 2: Extended Systems

- Politics and elections
- Crisis management
- Advanced economy
- More sophisticated AI

## Phase 3: Polish & Release

- Graphics and animations
- Sound and music
- UI polish
- Performance optimization
- Windows export

## Success Criteria for MVP

1. **Simulation runs stable** - 100 in-game days without crashes
2. **100+ citizens** - Manage at least 100 simulated citizens
3. **Economic flow** - Money flows between citizens, businesses, and government
4. **Needs impact** - Citizen satisfaction affects support rating
5. **Saveable** - Game can save and load state
6. **Playable Windows build** - Exports to .exe successfully
